FixtureFlow is a Python-based tournament fixture management software developed to organize and automate large-scale sports events at AIUB. It was successfully used in AIUB GAMES 2024, handling tournaments with over 3,000 student participants across multiple sports categories.
The system automatically calculates the total number of matches, manages byes, and generates structured round-wise fixture tables, reducing manual effort and scheduling errors. FixtureFlow ensures efficient, accurate, and scalable tournament organization for large events.

Technology: Python
Use Case: AIUB GAMES 2024 (https://www.aiub.edu/prize-giving-ceremony-of-aiub-games-2024---champions-reception)
